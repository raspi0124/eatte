# eatte
Collect attendance from Line app, reflect them to spreadsheet.  
LINEで出席欠席等を連絡すると自動的にスプレッドシートに反映させるBotのソースです。  
部活の出席管理用に作成したものです。  
こんな感じに自動反映させることができます。
